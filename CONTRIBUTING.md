# branchs

Toda branch  ser nomeada com as tags referentes ao seu tipo.

1 - feat: Nova funcionalidade ou adição de código no repositorio.

2 - fix: Correção de código ou bug não urgente.

3 - hotfix: Correção de código ou bug de caracter urgente, precisa ser resolvido imediato.

4 - preparação de uma nova versão oficial para produção, incluindo ajuste finais.
